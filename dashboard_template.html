Cómo mantener este panel actualizado (API de Mercado Público)

El panel calcula todo en el navegador a partir de un conjunto de datos embebido (instantánea del ). Para refrescarlo con nuevas órdenes de compra tienes dos caminos:

  1. Actualización manual: regeneras el archivo de datos desde tu Excel/consulta y reemplazas el bloque <script id="dataset">. Puedo entregarte el script que produce ese JSON.
  2. Automatización vía API ChileCompra: el endpoint es api.mercadopublico.cl/servicios/v1/publico/ordenesdecompra.json y exige un ticket (se solicita gratis en ChileCompra). Se consulta por día (fecha=ddmmaaaa), por estado, por CodigoOrganismo o CodigoProveedor. El navegador no puede llamarla directamente (CORS + el ticket no debe exponerse), así que se usa un pequeño proceso programado (cron / GitHub Actions) que consulta la API, filtra las OC de traslados, las anexa al histórico y regenera el JSON del panel.

Nota metodológica: el monto se expresa en pesos convertidos (MontoOC_BRUTO × ConversionRate), lo que lleva a CLP las órdenes emitidas en UF. La unidad de los indicadores es MM$ = millones de pesos.
